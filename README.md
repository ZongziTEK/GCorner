## Feature

This app will enable a feature similar to GNOME Hot Corner.

- When the program is running, the taskbar will automatically hide. 
- When the mouse hovers over the top left corner of the screen, the Windows task view will be opened and the taskbar will be displayed. 
- After the program exits, the taskbar will be restored.

## How to use

Download and execute it. Right click on the tray icon to exit.
